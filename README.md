## installation
Make sure you have **Nodejs** and **Git** installed on your machine. Then run the following commands
```cmd
mkdir instar-test
cd instar-test
git clone https://github.com/kiranharapradipta-gh/instar.git
cd instar/backend
notepad .env
npm install
node index
```
